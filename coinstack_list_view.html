Key Improvements

✓ Category Border

4px left border instantly shows Republic (red), Imperial (purple), Provincial (blue), Greek (olive).

✓ Reference Column

Primary catalog reference (Crawford, RIC, RPC) is now visible. Sub-label for mint or type name.

✓ Mint Column

Separate mint column for easy filtering and identification.

✓ Rarity Indicator

Colored dot + code (C, S, R1, R2, R3, U). "Unique" is rarity, not grade.

✓ Temperature Grades

Consistent color scale: blue (poor) → green (fine) → yellow (EF) → red (MS).

✓ Price + Paid

Shows both current value and acquisition cost. Useful for tracking without charts.

✓ Ruler Context

Sub-line shows reign dates or associated moneyers. More context at a glance.

✓ Thumbnail

Small obverse image for quick identification. Click to expand.

✓ Column Config

Toggle columns on/off. Add weight, diameter, die axis, provenance as needed.
